“CNN과 ResNet 기반 뇌출혈 예측 모델 비교 연구”


1. 프로젝트 개요
  1-1. 프로젝트 주제 및 선정 배경, 기획의도
      1) CNN과 ResNet 기반 뇌출혈 예측 모델 비교 연구
      2) 선정 배경
          - 뇌출혈은 초기 진단이 늦어질 경우 생명에 치명적 영향을 미칠 수 있으며, 빠른 진단과 치료가 중요하다.
	  - 의료 현장에서 CT, MRI 등의 영상 검사는 많이 사용되지만, 영상 판독에는 전문의의 경험과 시간이 필요하다.
	  - 최근 딥러닝 기반 이미지 분석 기술이 발전하면서, 의료 영상에서 질병을 자동으로 예측하고 보조 진단을 수행하는 연구가 활발히 진행되고 있다.
          - 따라서 뇌출혈을 빠르고 정확하게 예측할 수 있는 딥러닝 모델 개발이 필요하다.
      3) 기획의도.
      CNN 및 ResNet 계열 모델을 활용하여 뇌출혈 여부를 자동으로 예측할 수 있는 모델을 개발하고 ResNet50과 같은 전이학습 모델과, 
      Scratch로 학습한 모델을 비교하여 최적의 성능 모델을 탐색한다. 또 개발된 모델은 의료 영상 판독 보조 시스템으로 활용 가능성을 검토하며, 
      향후 조기 진단과 환자 치료에 기여할 수 있는 기반을 마련하고 작은 규모 데이터 환경에서의 모델 적용 가능성을 탐색하는 것을 목적으로 한다.

 1-2. 프로젝트 내용
	데이터 수집 및 전처리
	- 총 5,000장의 뇌 CT/MRI 영상을 수집하고, 라벨(뇌출혈 유무)을 정리함.
	- 이미지 크기 통일, 정규화, 증강(Augmentation) 등 모델 학습에 적합하도록 전처리 수행.

	모델 설계 및 학습
	- 기본 CNN 모델을 Scratch로 설계하여 학습을 진행함.
	- ResNet 구조를 Scratch로 구현하여 성능 비교.
	- ResNet50 전이학습(Pre-trained) 모델을 활용하여 학습 및 성능 평가.

	모델 평가 및 비교
	- CNN, Resnet of Scratch 모델과 Resnet50 전이학습 모델 간 성능 차이를 분석하여 최적 모델 선정.

	결과 활용 및 의미
	- 학습된 모델을 통해 뇌출혈 자동 예측 가능성을 검증.
	- 향후 의료 영상 판독 보조 시스템 개발 및 조기 진단 지원의 기초 자료로 활용 가능.


1-5. 활용방안 및 기대 효과

	활용방안
	- 개발된 뇌출혈 예측 모델을 의료 영상 판독 보조 시스템에 적용하여 전문의의 진단 시간을 단축하고, 진단 정확도를 향상시킬 수 있다.
	- 응급 상황에서 빠른 뇌출혈 여부 판단을 통해 신속한 치료 결정을 지원한다.
	- 제한된 데이터 환경에서도 딥러닝 모델이 적용 가능함을 보여주어 중소형 병원 및 연구기관에서도 활용 가능하다.

	기대 효과
	- 조기 진단과 치료 효과 향상: 뇌출혈 환자의 생존율과 회복률 개선에 기여.
	- 의료진 업무 부담 경감: 반복적인 영상 판독 업무를 보조함으로써 효율성 증대.
	- 딥러닝 기반 의료 영상 연구 확대: Scratch 학습과 전이학습 모델 비교 결과를 통해, 데이터 규모와 모델 선택에 대한 가이드라인 제공.




3. 프로젝트 수행 절차 및 방법
	문제 정의 및 목표 설정
	- 뇌출혈 자동 예측 모델 개발을 목표로 설정.
	- CNN, ResNet(Scratch), ResNet50 전이학습 모델을 비교하여 최적 모델 선정.

	데이터 수집 및 전처리
	- 5,000장의 뇌 CT/MRI 영상 수집 및 라벨링(뇌출혈 유무).
	- 이미지 크기 조정, 정규화, 데이터 증강(Augmentation) 수행.

	모델 설계 및 학습
	- CNN 모델: 기본 구조 설계 후 Scratch 학습.
	- ResNet 모델(Scratch): Residual Block을 활용한 모델 구현 및 학습.
	- ResNet50 전이학습: 사전 학습된 가중치 활용, Fine-tuning 수행.

	모델 평가 및 성능 비교
	- CNN, Resnet of Scratch 모델과 Resnet50 전이학습 모델 간 성능 차이 분석.

	결과 분석 및 활용 방안 도출
	- 학습 결과를 기반으로 최적 모델 선정.
	- 의료 영상 판독 보조 시스템 및 조기 진단 지원 가능성 평가.




4. 프로젝트 수행 경과

	1단계: 프로젝트 기획 및 목표 설정
	- 뇌출혈 자동 예측 모델 개발 필요성 분석
	- CNN, ResNet(Scratch), ResNet50 모델 비교 연구 계획 수립

	2단계: 데이터 수집 및 전처리
	- 5,000장 뇌 CT/MRI 영상 수집 및 라벨링
	- 이미지 정규화, 크기 조정, 데이터 증강(Augmentation) 수행

	3단계: 모델 설계 및 학습
	- CNN 모델 학습 진행
	- ResNet of Scratch 모델 구현 및 학습
	- ResNet50 전이학습 모델 학습

	4단계: 모델 평가 및 성능 비교
	- 모델별 장단점 분석 및 최적 모델 선정

	5단계: 결과 분석 및 활용 방안 검토
	- 학습 결과를 바탕으로 의료 영상 판독 보조 활용 가능성 평가
	- 조기 진단 및 환자 치료 개선 기여 가능성 도출


5. 자체 평가

	잘한 점
	- CNN과 ResNet 계열 모델을 모두 학습하여 모델 성능 비교 가능
	- 전이학습(ResNet50)을 활용하여 데이터가 제한적인 환경에서도 높은 성능 달성
	- 프로젝트 전 과정을 체계적으로 진행하여 데이터 수집, 전처리, 학습, 평가까지 완결

	개선점
	- 데이터 양(5,000장)이 상대적으로 제한적이어서 일부 모델 성능이 제한될 수 있음
	- 단일 기관/단일 장비 데이터 기반이라 일반화 가능성이 다소 낮음
	- 모델 해석 가능성(Explainability) 부분이 미흡하여 의료진 신뢰 확보가 필요

	향후 계획
	- 데이터 수를 늘리고 다양한 기관/장비의 영상을 포함하여 모델 일반화 향상
	- 추가적인 딥러닝 모델(Transformer 기반, EfficientNet 등) 비교를 통해 최적 모델 탐색 지속
